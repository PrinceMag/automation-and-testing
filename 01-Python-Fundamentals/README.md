# 01 — Python Fundamentals

> Learn to write Python from scratch — variables, data types, collections, control flow, functions, and basic OOP.

---

## Course Overview

Python is the most in-demand programming language in the world. This course gives you a solid foundation in Python so that you can write real programs, understand code, and move confidently into automation.

**Duration:** 4 weeks  
**Effort:** 5–10 hours per week  
**Level:** Complete beginner  
**Pre-requisite:** None

---

## What You Will Learn

- Create and use variables with the correct data types
- Work with strings, integers, floats, and booleans
- Use collections: lists, tuples, sets, and dictionaries
- Control program flow with `if/elif/else` and loops
- Write reusable functions and understand scope
- Import and use Python modules
- Read and write files
- Understand the basics of Object-Oriented Programming (OOP)
- Handle errors with `try/except`

## 4-Week Roadmap

### Week 1 — Variables, Data Types & Strings

| Topic | Key Concepts |
|-------|-------------|
| Variables | Naming rules, assignment operator, dynamic typing |
| Data types | `str`, `int`, `float`, `bool` |
| String operations | `.upper()`, `.lower()`, `.replace()`, `.strip()`, `.split()`, `len()`, slicing, f-strings |
| Type casting | `int()`, `float()`, `str()`, `bool()`, `type()` |

### Week 2 — Collections

| Topic | Key Concepts |
|-------|-------------|
| Lists | Ordered, mutable, indexing, `.append()`, `.remove()`, `.sort()`, `.pop()` |
| Tuples | Ordered, immutable, unpacking, use cases |
| Sets | Unique values, `.add()`, `.union()`, `.intersection()`, `.difference()` |
| Dictionaries | Key-value pairs, `.get()`, `.keys()`, `.values()`, `.items()`, nested dicts |

### Week 3 — Control Flow & Functions

| Topic | Key Concepts |
|-------|-------------|
| Conditionals | `if`, `elif`, `else`, comparison operators, logical operators |
| Loops | `for`, `while`, `break`, `continue`, `pass`, `range()` |
| Functions | `def`, parameters, return values, `*args`, `**kwargs` |
| Scope | LEGB rule — Local, Enclosing, Global, Built-in |
| Lambda | Anonymous functions, when to use them |

### Week 4 — Modules, Files & OOP Basics

| Topic | Key Concepts |
|-------|-------------|
| Modules | `import`, `from...import`, pip, `math`, `random`, `datetime`, `os` |
| File handling | `open()`, read/write/append modes, `with` statement |
| JSON | `json.loads()`, `json.dumps()`, reading JSON files |
| OOP | `class`, `__init__`, `self`, methods, inheritance |
| Error handling | `try`, `except`, `else`, `finally`, common exception types |

## Key Python Rules to Memorise

```python
# Variable naming rules
my_name = "valid"    # ✅ letters, numbers, underscores
2name = "invalid"    # ❌ cannot start with a number
my-name = "invalid"  # ❌ hyphens not allowed
class = "invalid"    # ❌ reserved keyword

# Python is case-sensitive
name = "Thandi"
Name = "Sipho"       # These are two different variables

# Dynamic typing — no need to declare types
x = 10        # int
x = "hello"   # now a str — perfectly valid

# Type casting
age = int("21")       # str → int
price = float("9.99") # str → float
label = str(42)       # int → str
```
