# Week 1 — Variables and Data Types

## Topics Covered

| File | Topic |
|------|-------|
| `01_variables.py` | Variable naming rules and Python naming conventions |
| `02_string_datatype.py` | String creation, multi-line strings, and user input |
| `03_numeric_datatypes.py` | Integer and float — arithmetic and operations |
| `04_boolean_datatype.py` | Boolean type — True/False and real-world use cases |
| `05_type_casting.py` | Converting between types: int(), str(), float(), type() |
| `06_scope_and_structure.py` | Local vs global scope and Python code structure |

## How to Run a File

```bash
python 01_variables.py
```

Or open any file in VS Code and press **F5** to run it.

## Learning Objectives

By the end of this week you should be able to:

- Create a variable and assign a value to it
- Name variables correctly following Python conventions
- Identify the four core data types: `str`, `int`, `float`, `bool`
- Convert between data types using casting functions
- Explain the difference between local and global scope
