# Automation And Testing

> A structured 4-month curriculum covering Python, Java, and Selenium automation — built for beginners aiming for a career in QA Automation Engineering.

---

## About This Repository

This repository contains all course material, notes, assignments, and solutions for the **Automation And Testing** programme at **Zinza Institute Of Technology**.

The programme takes a complete beginner from writing their first Python variable all the way to building a professional Java Selenium test suite using the Page Object Model.

---

## Programme Structure

| # | Course | Duration | Language | Focus |
|---|--------|----------|----------|-------|
| 01 | [Python Fundamentals](./01-Python-Fundamentals/) | 4 weeks | Python | Core programming concepts |
| 02 | [Python Automation with Selenium](./02-Python-Automation-Selenium/) | 4 weeks | Python | Browser automation + pytest |
| 03 | [Java Fundamentals](./03-Java-Fundamentals/) | 4 weeks | Java | OOP and typed programming |
| 04 | [Java Automation with Selenium](./04-Java-Automation-Selenium/) | 4 weeks | Java | TestNG + Page Object Model |

**Total duration:** 4 months (16 weeks)  
**Effort:** 5–10 hours per week  
**Level:** Beginner → Job-ready

## How to Use This Repo

1. Start with **01-Python-Fundamentals** — read the README, follow the roadmap week by week.
2. Complete each assignment before moving to the next course.
3. Check the `assignments/` folder for solutions — only look after attempting the work yourself.
4. Use [W3Schools Python](https://www.w3schools.com/python/) and [W3Schools Java](https://www.w3schools.com/java/) as your reference while studying.
